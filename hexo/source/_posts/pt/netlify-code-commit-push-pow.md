A algum tempo fale aqui sobre o surge, uma ferramenta bem legal para fazer deploy de forma rápida e prática de site estáticos e/ou aplicações front-end. Ele resolve o problema de forma bem aceitável, mas possui limitações. Por isso decidi pesquisar por alguma solução mais completa, achei plausível, uma vez que aplicações que rodam somente no client estão em alta com as SPA (single page applications) e PWA (progressive web apps).

Encontrei o Netlify! E preciso dizer: que ferramenta legal! Ela faz tudo e mais um pouco para você não ter que se preocupar com a infra da sua aplicação. Pra começar, basta criar um novo projeto é conectar ao seu repositório git na sua plataforma de preferência como, Github, Bitbucket e Gitlab. Em seguida você pode escolher o branch que será usado para deploy, assim como o comando de build que deseja para o deploy e a pasta que deve ficar publica no seu domínio. Clicar no botão de “publish” e ser feliz.
